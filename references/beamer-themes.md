# Beamer Themes

## Supported Themes

### Berlin

Default academic theme for this skill.

Use when:

- the user does not specify a theme
- the deck should feel conventionally academic
- structure and readability matter more than visual novelty

### Madrid

Use when:

- the user wants a slightly cleaner and lighter look
- the deck needs a conventional conference feel

### Boadilla

Use when:

- the user wants a soft, minimal academic tone
- the deck is text-heavy and should feel less visually dense

### AnnArbor

Use when:

- the user wants a stronger header presence
- the presentation should feel more formal or institutional

### metropolis

Use when:

- the user wants a more modern Beamer look
- the deck benefits from a cleaner contemporary feel

## Default Rule

If the user does not choose a theme, use `Berlin`.

## Theme Preset Assets

Theme snippets live in `assets/theme-presets/` and should define:

- theme name
- optional color overrides
- any title page or footer conventions
- optional font theme overrides
